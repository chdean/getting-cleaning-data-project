# README

*run_analysis.R* downloads the [UCI Human Activity Recognition Using Smartphones Dataset][dataset] and performs basic cleaning and analysis.  The output *tidyData.txt* contains the mean sensor data separated by subject and activity.

Refer to CodeBook.md for details about the transformations performed and output.  The README and features.txt files included with the dataset give more details about the data itself.

[dataset]: http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones
